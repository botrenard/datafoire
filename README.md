# Qu’est ce que c’est ?

La data-foire est le point de contact d’un réseau d’échange de fichiers
numériques hors-Internet. Les données du réseau data foires se
déplacent avec les participant•es, dans des clefs usb dans leurs poches,
et se brassent au sein des stands de data foires.

Pour en savoir plus, regardez le PDF !
